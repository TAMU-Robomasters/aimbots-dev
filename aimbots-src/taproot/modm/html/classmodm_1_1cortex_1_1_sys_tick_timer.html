void modm::platform::SysTickTimer::disable ( )
static

Disables SysTick Timer.

Warning
If the SysTick Timer is disabled modm::Clock is not incremented automatically. Workflow classes which rely on modm::Clock will not work if modm::Clock is not incremented.

void modm::platform::SysTickTimer::initialize ( )
inlinestatic

Initializes the SysTick Timer to generate periodic events.

Warning
The SysTick Timer is used by default to increment modm::Clock, which is used by modm::Timeout and other similar processing classes. You must not increment the modm::Clock additionally somewhere else.
