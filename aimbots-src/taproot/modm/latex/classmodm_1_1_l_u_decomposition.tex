\doxysubsection{Detailed Description}
Class for decomposing matrices. 

Factorise a matrix A into an L(ower) and U(pper) matrix such that A = L\texorpdfstring{$\ast$}{*}U or P\texorpdfstring{$\ast$}{*}A = L\texorpdfstring{$\ast$}{*}U where P is a pivot matrix (changes the row order).

Adapted from the implementation of Gaspard Petit (\href{mailto:gaspardpetit@gmail.com}{\texttt{gaspardpetit@gmail.\+com}}).
