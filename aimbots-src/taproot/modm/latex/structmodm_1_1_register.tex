\subsubsection*{template$<$typename T$>$\newline
struct modm::\+\+Register$<$ T $>$}
Class for holding a register value.

You can cast any \doxylink{structmodm_1_1_flags}{Flags}, \doxylink{structmodm_1_1_configuration}{Configuration} or \doxylink{structmodm_1_1_value}{Value} type into the approprate \doxylink{structmodm_1_1_register}{Register} type.

Apart from the boolean operators, this base class deletes all other implicit operator overloadings. Since any type information is lost during casting, you should not be able to use any operations involving specific flags, configurations or values.

Instead you have to explicitly use the integral {\ttfamily \doxylink{structmodm_1_1_register_a4d86255f7cd8fc09c9f88d1776535f68}{value}} to perform your potentially unsafe operations.

This class makes heavy use of {\ttfamily constexpr}, so that as many values and operation can take place at compile time for efficiency.
