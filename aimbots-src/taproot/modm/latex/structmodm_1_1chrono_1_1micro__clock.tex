\doxysubsection{Detailed Description}
Returns a time\+\_\+point in microseconds.

This clock is meant for measuring durations (time differences), not time since reboot. You must not depend on this clock for timekeeping, especially not over long periods of time, since:\+


\begin{DoxyEnumerate}
\item the clock is NOT STEADY! The 32-\/bit microsecond counter will wrap around after 71 minutes!
\item the clock may have quite some drift and is not synchable to an external reference. You must therefore not rely on this clock for very long durations that need to be {\itshape accurate}.
\end{DoxyEnumerate}

You must use an actual calendar implementation for proper time keeping!
