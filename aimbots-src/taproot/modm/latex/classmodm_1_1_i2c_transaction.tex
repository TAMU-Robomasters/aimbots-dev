\doxysubsection{Detailed Description}
Abstract class for transactions.

For true asynchronous operation, the communication driver should inherit from this class, allowing multi-\/stage driver design and performance gain by premature data evaluation. This transaction object will stay attached to the {\ttfamily \doxylink{classmodm_1_1_i2c_master}{I2c\+Master}} during whole operation.
