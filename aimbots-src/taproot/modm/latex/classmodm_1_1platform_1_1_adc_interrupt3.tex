\doxysubsection{Detailed Description}
ADC \doxylink{classmodm_1_1platform_1_1_adc3_a62e08deee4e0f7c4ba0b539164f09a80}{Interrupt} module

This class allows you to attach functions to the ADC Conversion Complete \doxylink{classmodm_1_1platform_1_1_adc3_a62e08deee4e0f7c4ba0b539164f09a80}{Interrupt} via function pointers. Be aware however, that this implementation is slower and requires more resources than writing the function code directly into the interrupt service routines.
