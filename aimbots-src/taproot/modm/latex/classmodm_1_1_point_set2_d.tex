\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class modm::\+\+Point\+Set2\+D$<$ T $>$}
Point set. 

Collection of points, represented by their corresponding vectors. Used for example to hold the result of a intersection-\/operation.

Based on the \doxylink{classmodm_1_1_dynamic_array}{modm::\+\+Dynamic\+Array} class, therefore grows automatically if more space than currently allocated is needed. But because this is an expensive operation it should be avoid if possible.
