The {\ttfamily \doxylink{classmodm_1_1log_1_1_logger}{modm::\+log::\+\+Logger}} uses a {\ttfamily \doxylink{classmodm_1_1_i_o_stream}{modm::\+\+IOStream}} to read messages and write them to a given output device.

Write a log message with streams:\+


\begin{DoxyCode}{0}
\DoxyCodeLine{MODM\_LOG\_DEBUG\ <<\ \textcolor{stringliteral}{"{}i="{}}\ <<\ i\ <<\ \textcolor{stringliteral}{"{},\ y="{}}\ <<\ y\ <<\ modm::flush;}

\end{DoxyCode}


The name of the stream represents the log level of the message. Available are:\+


\begin{DoxyItemize}
\item {\ttfamily MODM\+\_\+\+LOG\+\_\+\+DEBUG}
\item {\ttfamily MODM\+\_\+\+LOG\+\_\+\+INFO}
\item {\ttfamily MODM\+\_\+\+LOG\+\_\+\+WARNING}
\item {\ttfamily MODM\+\_\+\+LOG\+\_\+\+ERROR}
\end{DoxyItemize}
