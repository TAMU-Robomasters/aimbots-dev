class modm::\+allocator::\+\+Block$<$ T, BLOCKSIZE $>$}
\doxylink{classmodm_1_1allocator_1_1_block}{Block} allocator. 

Allocates a big block of memory and then distribute small pieces of it. The memory is not released until the destruction of the allocator. If more memory is needed a new block is allocated.

This technique is known as "{}memory pool"{}.
