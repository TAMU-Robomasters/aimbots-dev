\doxysubsection{Detailed Description}
Container which destroys itself when the last copy is destroyed. 

This container saves a copy of the given data on the heap. It provides the functionality of a shared pointer =\texorpdfstring{$>$}{>} pointer object records when it is copied -\/ when the last copy is destroyed the memory is released.
