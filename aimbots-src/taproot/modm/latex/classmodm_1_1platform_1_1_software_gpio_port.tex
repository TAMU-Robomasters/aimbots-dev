\doxysubsection{Detailed Description}
\subsubsection*{template$<$class... Gpios$>$\newline
class modm::\+platform::\+\+Software\+Gpio\+Port$<$ Gpios $>$}
Create an up to 32-\/bit port from arbitrary pins.

This class optimizes the data type for the {\ttfamily read()} and {\ttfamily write()} methods. Supplying up to 8 Gpios will use {\ttfamily uint8\+\_\+t}, up to 16 Gpios {\ttfamily uint16\+\_\+t} and up to 32 Gpios {\ttfamily uint32\+\_\+t}.

\begin{DoxyNote}{Note}
Since the bit order is explicitly given by the order of template arguments, this class only supports {\ttfamily \doxylink{classmodm_1_1_gpio_port_a7ff5dde1f0e7851f0059d35be50bea33a960b44c579bc2f6818d2daaf9e4c16f0}{Data\+Order::\+\+Normal}}. If you need reverse bit order, reverse the order of {\ttfamily Gpios}!
\end{DoxyNote}
