\doxysubsection{Detailed Description}
ADC \doxylink{classmodm_1_1platform_1_1_adc1_af3e3e6935d529268f47c5e77b0ce46ab}{Interrupt} module

This class allows you to attach functions to the ADC Conversion Complete \doxylink{classmodm_1_1platform_1_1_adc1_af3e3e6935d529268f47c5e77b0ce46ab}{Interrupt} via function pointers. Be aware however, that this implementation is slower and requires more resources than writing the function code directly into the interrupt service routines.
