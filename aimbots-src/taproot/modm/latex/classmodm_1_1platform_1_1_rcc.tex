\doxysubsection{Detailed Description}
Reset and Clock Control for STM32 devices.

This class abstracts access to clock settings on the STM32. You need to use this class to enable internal and external clock sources \& outputs, set PLL parameters and AHB \& APB prescalers. Don\textquotesingle{}t forget to set the flash latencies.
