\subsubsection*{template$<$typename T, std::\+size\+\_\+t N$>$\newline
class modm::\+\+Bounded\+Deque$<$ T, N $>$}
Double ended queue. 

Internally organized as a ring buffer.

\begin{DoxyVerb}*           tail --\                 /-- head
*                  |                 |
*        +------+------+---- ----+------+------+
*      0 |      | data |   ...   | data |      | N-1
*        +------+------+---- ----+------+------+
*           |      |                 |      |
* prepend --/      |                 |      \-- append
*                  |                 |
*       getFront --/                 \-- getBack
* \end{DoxyVerb}



\begin{DoxyTemplParams}{Template Parameters}
{\em T} & Type of the elements \\
\hline
{\em N} & Size of the queue\\
\hline
\end{DoxyTemplParams}
Up to a size of 254 small index variables with 8-\/bits are used, after this they are switched to 16-\/bit.

\begin{DoxyWarning}{Warning}
This class don\textquotesingle{}t check if the container is empty before a pop-\/operation. You have to do this by yourself!
\end{DoxyWarning}
