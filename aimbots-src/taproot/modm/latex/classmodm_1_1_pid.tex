\subsubsection*{template$<$typename T, unsigned int Scale\+Factor = 1$>$\newline
class modm::\+\+Pid$<$ T, Scale\+Factor $>$}
A proportional-\/integral-\/derivative controller (PID controller). 

The PID controller is one of the basic controlling algorithm.

The {\ttfamily max\+Error\+Sum} is to be used, to limit the internal integrator and so provide an anti wind up.

With the template parameter {\ttfamily Scale\+Factor} this class provides an fix point capability with integer types.

Example for a motor speed control with a 10-\/bit PWM output.
